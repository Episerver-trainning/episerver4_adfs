Search User/Groups

If you wish to check which groups a particular individual is a member of, other settings or maybe give a user a new password, you must find the user’s account. This can be done using the Search for user/group option. Specify the type (group or user) you want to search for; specify search criteria in one or more of the other fields and select Search. If you want to list all groups and users, leave all the fields empty, select type All and select Search.

The search results are listed and by clicking a group or user name, you can edit the information about that group/user. In the Number of hits per page field, the number of users should be displayed on each page. If the search results produce several pages scroll through the pages in the lower left corner.
