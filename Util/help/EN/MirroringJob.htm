Mirroring Service

Using the Mirroring Service option, you can set how often the system is to activate the mirroring of contents for other Web sites. If your Web site is set up to mirror content from one Web site to another it can be carried out by a single editor or automatically by the system. If the system is selected to mirror the information use the Mirroring Service function to set when and how often the system is to activate mirroring.
