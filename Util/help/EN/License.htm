License Information

The license for the EPiServer is connected to an IP address or MAC address on the machine on which it is installed. The license information for your EPiServer Web site is stored in an encrypted file on the Web server. This information can tell you to whom the license belongs, what type of license it is and if there are any relevant restrictions. If for any reason you need to change your license: if, for example, you change the IP address or type of license, etc., you can upload and install the new license file from the Install a new license tab.
