Edit Language Codes

EPiServer is capable of managing all types of language. There are eight languages as standard, Swedish, Norwegian, Danish, Finnish, German, French, Spanish and English. If your organization requires further languages, it is possible to add these. By adding new languages under Edit language codes, you can add extra support for language encoding of an individual page. If a page has been language encoded, all standard texts on the page are in the selected language, provided the language has been translated. If you also require local settings such as time format, currency etc. to be displayed in the applicable language, it must be listed under Edit language codes.

In the example below we want to add New Zealand as a new language. They speak English there, so it may feel unnecessary. But at the same time they write the date differently, compared to other English-speaking countries. Therefore we want to be able to choose to have the website according to New Zealand standards.

Add a new language as follows:

  1. Select Edit language codes on the Config tab.
  2. Select Add file.
  3. Provide an abbreviation for the name. This should also be used later in the XML files.
  4. Provide a name for the language.
  5. State which regionalism the language will have. This is what controls how the date, time, etc. is shown.
  6. Select Save.
