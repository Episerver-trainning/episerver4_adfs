Manage Web Site Languages

If you want your editors to be able to write contents in a particular language, the language must be added, and activated for use. You do this under the Manage Web site languages option.

Here you can also decide which language is to be the default language on your Web site. The default language is the language that shall apply first when a visitor arrives at the Web site. Normally you set the language to be one that you believe the majority of your target group wish to read. If the visitor wants another language, an active action is required from them.

When you open Manage Web Site languages, all added languages are shown in a long list. Straight away you can see which languages are available and if they are active or not. If you wish to change the settings for a language, click on the language’s name. You can adjust the order of the various languages using the arrows to the left of the language.

Add new language for multilingual Web sites

For an editor to be able to create contents in several different languages, the language must be added under Manage Web site languages and be active.

To add a new language that will be accessible for contents management, do the following:

  1. Select Manage Web site languages on the Config tab.
  2. Select Add language.
  3. In the Name field, provide a name for that language. It is this name that will be shown to the editor.
  4. Mark Active if you want the language to be active for editing.
  5. In the Language codes field, provide the language code that will apply for that language. You have to have added the language under Edit language for it to be shown in the list.
  6. In the Icon field, provide the search path to an icon that will symbolize the language to the editor. There are a number of flags stored in the \MyEPiServer\Util\images\Flags directory, you are free to use these. If you are going to make your own icons, these should measure 15x15 pixels so that they fit properly in the edit mode.
  7. In the Prefix for Web site field, you can provide a specific prefix to be able to easily show the contents of the relevant language. If you do not provide a prefix, the language’s language code will be used as a prefix, for example www.foretaget.se/nz.
  8. In the Sortings order field, you can provide a number for the language’s placement in the list. You can also control this with the help of arrows when you have saved the language.
  9. Select Save.
